# AI Company Platform

Monorepo in Desktop with:
- backend (Express + SQLite)
- frontend (React + Vite)

## Features scaffolded
- Auth: register/login
- Company/Site/Team/Project CRUD
- Agent CRUD + prompt template update
- Kanban: boards, columns, cards, move cards
- Run from card (PM/DEV/QA/OPS pipeline)
- Message logs + artifacts per run
- LLM provider config entities (OpenAI + Ollama)
- GitHub integration scaffold endpoints

```bash
cd /Users/jayeshlodhiya/Desktop/ai-company-platform
```
